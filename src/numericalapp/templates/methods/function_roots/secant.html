Secant Method

The objective of this method is to find a root of a function from two initial values, a tolerance and a number of iterations, in this case it is not necessary to have an interval. The secant method is defined as a variant of Newton's method. From the iterative equation that defines Newton's method, the derivative is replaced by an expression that approximates it: X2 = X1 - ((f (X1) * (X1-Xo)) / (f (X1) -f ( Xo))

1. You must choose two initial approximations X1 and X0

2. X2 = Expression is calculated, Xn = Expression (n-1)

3. And the previous step is repeated until reaching an approximation.



NOTE: You need to make sure that the function in continuous for the given interval.
Replace ^ with **
Example: x^2 = x**2


  • Tolerance must have a positive value.
  • The iteration number must be positive.
  • The function must be continuous and differentiable.
  • Be sure the derivatives of the function are correct.
  • For the method to be faster the initial values must be close to the root
